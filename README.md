
# view app files
```
run-as com.sample.demo_keystore
```
This automatically changes directory into `/data/user/0/com.sample.demo_keystore/files`


# manually adding castle sdk

libs dir should look like:
```
app/libs/
├── AndroidManifest.xml
├── highwind-1.5.0.aar
├── META-INF
│   └── com
│       └── android
│           └── build
│               └── gradle
│                   └── aar-metadata.properties
├── proguard.txt
└── R.txt
```
